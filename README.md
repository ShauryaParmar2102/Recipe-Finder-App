# Recipe-Search-App

A web application that allows users to search for meals and discover random recipes using TheMealDB API. Users can enter a keyword to find recipes or generate a random meal idea with a single click.

## 🔧 Features
Search recipes by name
Fetch random recipe on demand
Displays meal image and title
Loading and error messages for better UX
Clean, responsive UI layout

## 🚀 Tech Stack

- HTML

- CSS

- JavaScript

- TheMealDB API

## ⬇️ How to Download / Run

1: Download ZIP

2: Click the green “Code” button on GitHub

3: Select “Download ZIP”

4: Extract the folder

5: Open index.html in your browser (double-click it) or use live server
