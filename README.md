# Recipe-Search-App

A simple and interactive web application that allows users to search for recipes or generate random meal ideas using TheMealDB API. The goal of this project is to practice working with APIs, asynchronous JavaScript, and DOM manipulation while building a clean and user-friendly interface.

This project simulates a real-world frontend application where users can either search for a specific dish or discover something new with a random recipe generator.

## 🔧 Features

🔍 Search for recipes by name (e.g. chicken, pasta, burger)

🎲 Generate a completely random recipe with one click

🖼️ Displays meal image, name, and basic recipe data

⏳ Loading state while fetching data from API

⚠️ Error handling for network issues or invalid searches

🚫 Friendly message when no results are found

## 🛠️ Tech Stack

- HTML

- CSS

- JavaScript

- TheMealDB API

1: Download ZIP

2: Click the green “Code” button on GitHub
